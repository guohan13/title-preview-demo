# [OPEN] phone-preview-access

## 问题描述
- 现象：手机上无法预览当前页面。
- 目标：确认是访问地址错误、服务监听异常、局域网访问受阻，还是页面本身报错导致手机不可用。

## 已知环境
- 项目目录：`/Users/bytedance/Desktop/design demo`
- 预览服务：`sync_server.py`
- 当前尝试端口：`8001`

## 初始假设
- H1：手机访问的还是旧端口 `8000`，而当前可用服务实际跑在 `8001`。
- H2：本地服务虽然在电脑上可访问，但没有正确监听到局域网接口，手机访问不到。
- H3：电脑和手机不在同一个局域网，导致 `10.255.75.13:8001` 无法互通。
- H4：Mac 防火墙或系统权限阻止了 Python 对外提供 `8001` 访问。
- H5：页面脚本还有残留调试请求或运行时错误，导致手机端打开后白屏或加载失败。
